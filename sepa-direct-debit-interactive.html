SEPA Direct Debit — B2B (SDD B2B)

D-1 ALL SEQ TYPES · pain.008 (B2BB) · NO REFUND RIGHT · DEBTOR BANK VALIDATES MANDATE · CORPORATE ONLY

⚠ Key B2B differences vs Core: (1) Debtor bank must hold and validate the mandate before each debit. (2) No unconditional refund right. (3) Shorter return window: D+2. (4) Corporate debtors only. (5) D-1 cut-off for all sequence types.

SDD Core

  • B2C + B2B debtors allowed
  • Debtor bank: no prior mandate check
  • 8-week unconditional refund right
  • Return window: D+5 business days
  • FRST/OOFF D-1, RCUR/FNAL D-2 cut-off
  • Lower creditor liability

SDD B2B

  • Non-consumer (corporate) debtors only
  • Debtor bank holds & validates mandate
  • No unconditional refund right
  • Return window: D+2 business days
  • FRST/RCUR/FNAL/OOFF: D-1 cut-off
  • Higher creditor control / lower dispute risk
